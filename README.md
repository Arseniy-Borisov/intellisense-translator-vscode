# IntelliSense Translator for VSCode

**Перевод hover‑подсказок через локальную модель**  
Используется модель Ollama `ali6parmak/hy-mt1.5:1.8b`.

## Особенности

- Работает **только при наведении мышью**.  
  Подсказки при вводе с клавиатуры не поддерживаются из-за особенностей взаимодействия VSCode API и TS Server.
- Поддерживается **33 языка**.
- Язык по умолчанию — **русский**.

## Установка

> Добавьте инструкцию по установке, если требуется.

## Использование

1. Наведите мышь на элемент кода.
2. Появится переведённая подсказка на выбранном языке.
